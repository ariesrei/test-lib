# STEPS

1. Install Docker 
	-$ ubuntu / windows 10
	-$ docker -v (version of docker)

	Create / Generate:
		Dockerfile
		docker-compose.yml

2. Install NPM 
	-$ npm -v (version on npm)
	-$ npm install --save-dev [packagename] ( save the package in node_mudules folder )
	-$ npm init -y (generate package.json)

	Create / Generate: 
		package.json


3. Install RollUp JS
	-$ npm install --save-dev rollup
	( rollup plugins, some are optional )
	-$ npm install --save-dev rollup-plugin-commonjs
	-$ npm install --save-dev rollup-plugin-node-resolve
	-$ npm install --save-dev rollup-plugin-json
	-$ npm install --save-dev rollup-plugin-babel
	-$ npm install --save-dev rollup-plugin-browsersync

	Create / Generate:
		rollup.config.js


		export default {
  			input: 'src/main.js',
  			output: {
    				file: 'app/js/bundle.js',
				format: 'cjs'
		  	},
		  	plugins: [
		  		json(),
			  	babel({
			  		exclude: 'node_modules/**',
			  	}),
			  	commonjs(),
			  	resolve({
			  		jsnext: true,
			  		main: true,
			  		browser: true
		  		}),
			    	browsersync({
			      		server: 'app',
				      	browser: ["google chrome", "firefox"],
				      	open: false
			    	}),
		  	],
			};


4. Install Babel
	-$ npm install --save-dev babel
	( babel requirements)
	-$ npm install --save-dev babel-core ( prerequisite )


5. Install Karma


5. Update package.json


	"scripts": {
    		"test": "echo \"Error: no test specified\" && exit 1",
    		"build": "rollup -c"
  		},

5. $
