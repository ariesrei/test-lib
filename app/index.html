Email link converter

What is email link converter?

Email converter is a simple js scirpt that transforms an element to an Email link format ( mailto:email@domain.com ). One purpose of this script is to avoid email crawlers, email scraping tools to grab your email and send you spam.

How it works?

Just add the following on the Element.

  1. A className : class="emailTo".
  2. A data url : data-url="your url".

Email convert uses class attributes so that it can get all instances of the class "emailTo".


example: 
 <div data-url="http://www.google.com" class="emailTo">iamemail</div> (source code)

Converted to: 

 iamemail@google.com (page displays) - <a href="mailto:iamemail@google.com"> iamemail@google.com </a>
